## HTML
Hyper Text Markup Language <br>
It's used for defining layout of a web page.
### A basic HTML page 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Our First Website</title>
</head>
<body>
</body>
</html>
